# 登录/注册页面设计文档

> 日期：2024-03-12
> 状态：待实现

## 1. 需求概述

为教育场景演示项目设计并开发登录、注册页面，支持：
- 手机号/用户名登录
- 学生/老师/家长三类快捷演示登录
- 手机号注册流程（验证码、密码设置）

### 2.5 交互反馈

**输入验证：**
- 正常：`input` 默认样式
- 获取焦点：`input-primary`
- 输入正确：`input-success` + ✓
- 输入错误：`input-error` + 错误提示

**验证码倒计时：**
- 点击后显示 "60s 后重发"
- 倒计时期间按钮禁用
- 结束后显示 "重新获取"

**密码校验：**
- 长度：6-20 位
- 实时提示"至少6个字符"
- 确认密码：失焦校验一致性

**操作反馈：**
- 成功：`alert-success`
- 失败：`alert-error`
- 加载中：按钮显示 loading 状态
